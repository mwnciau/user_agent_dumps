# Top User Agents

See [microlinkhq/top-user-agents](https://github.com/microlinkhq/top-user-agents).

> An always up-to-date list of the top 100 HTTP user-agent strings most used over the Internet.

The historic list of user agents in the above repository is downloaded and aggregated here.

## Downloading the latest data

Requirements:

* Node.js
* A GitHub token

To download the latest data, run:

```bash
node download_and_agregate.js
```
